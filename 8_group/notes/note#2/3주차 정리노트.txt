Q1 : 왜 머신러닝 할 때 데이터 정리하고 훑어봐야하는지?
은미 : 정제된 데이터를 써야해서. 결과의 정확도를 높이기 위해서
승현 : 데이터를 미리 확인하지 않고 사용하면 이상한 결과값이 나올 수 있으니까
그걸 미리 정제를 하기 위해서 같다.


Q2 : 테스트 세트와 훈련 세트
은미 : 훈련, 테스트 세트 없이 할 수 있는가?
승현 : 우선 테스트 세트랑 훈련 세트가 정확히 뭔지 알아봐야겠다
=> 검색 결과, 훈련 세트는 연습시킬 때 사용하는 데이터이고
테스트 세트는 실제 모델 테스트를 할 사용하는 데이터


Q3 : 무작위 샘플링 vs 계층적 샘플링
은미 : 둘 다 어떤 장단점이 있나?
승현 : 우선 계층적 샘플링은..
은미 : 무작위 샘플링보다 계층적 샘플링이 더 정확한 결과를 얻을 수 있을 것 같다?
승현 : 그럼 무작위 샘프링의 장점이 뭘까요
계층적 샘플링보다 다양한 결과값을 얻을 수 있을 것 같다?
은미 : 결론은 계층적 샘플링을 사용하는게 더 나을 것 같다
